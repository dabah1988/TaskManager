🚀 Présentation de l'application Task Manager

Task Manager est une application web de gestion de projets et de tâches, conçue pour aider les équipes à organiser efficacement leur travail, suivre l'avancement des projets et gagner en productivité.

Développée avec ASP.NET Core en backend et Angular pour l’interface utilisateur, cette application illustre les principes de la Clean Architecture et de la programmation orientée services.

🔧 Fonctionnalités principales

  • ✅ Gestion des projets : création, modification, suppression de projets avec suivi du statut.
  • 📋 Gestion des tâches : chaque projet peut contenir plusieurs tâches assignées à des membres de l'équipe.
  • 👥 Affectation des utilisateurs aux projets et tâches.
  • 📈 Tableau de bord dynamique : vue d’ensemble sur l’avancement des projets en temps réel.
  • 🔍 Recherche et filtres intelligents pour retrouver rapidement les informations.
  • 🛠️ Tests unitaires et d’intégration avec mocks pour assurer la fiabilité du code.
  • ☁️ Déploiement avec Docker et compatible avec une architecture de microservices.

🧱 Architecture technique

  • ASP.NET Core Web API pour l’API RESTful
  • Entity Framework Core (Code First)
  • Angular 18 pour l’interface utilisateur
  • RxJS & Observables pour les flux asynchrones
  • Pattern Repository & Service pour une architecture claire
  • Swagger pour la documentation d'API
  • Serilog pour le logging structuré
